# GitHub Repository Intelligence Dashboard

A Flask web application that analyzes public GitHub repositories.

## Features

- Repository Details
- Stars
- Forks
- Watchers
- Open Issues
- Primary Language
- Created Date
- Updated Date

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- GitHub REST API

## Run

```bash
pip install -r requirements.txt
python app.py
```
